HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. No es un lenguaje de programación, sino un conjunto de etiquetas que se usan para estructurar el contenido de una página, como textos, imágenes, enlaces, tablas, formularios, y más. Con HTML, le dices al navegador qué mostrar y cómo está organizado el contenido, aunque el diseño visual generalmente se logra con CSS (Hojas de Estilo en Cascada).

Algunas Etiquetas Básicas de HTML

HTML se compone de etiquetas que definen el tipo de contenido que se mostrará en la página. Aquí te dejamos algunas de las más comunes:

  • <h1>...</h1> a <h6>...</h6>: Se usan para los títulos. <h1> es el más importante, y <h6> el menos.
  • <p>...</p>: Define un párrafo de texto.
  • <a href="...">...</a>: Crea un enlace que te lleva a otra página o sitio web.
  • <img src="..." alt="...">: Inserta una imagen.
  • <div>...</div>: Se usa como contenedor general para agrupar otros elementos.
  • <span>...</span>: Similar a <div>, pero para contenido en línea.

IA

La Inteligencia Artificial (IA) es una rama de la tecnología que busca crear sistemas capaces de imitar la inteligencia humana. Esto incluye tareas como entender lenguaje, aprender, resolver problemas, reconocer imágenes o tomar decisiones.
